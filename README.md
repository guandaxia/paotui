# paotui

> A Vue.js project

基于`Vue.js`开发的项目

`UI`使用 `Vux`

主要功能有： 微信支付、分享

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build

# build for production and view the bundle analyzer report
npm run build --report

# run unit tests
npm run unit

# run e2e tests
npm run e2e

# run all tests
npm test
```
